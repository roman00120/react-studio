# Lumina Studio — Landing Page SaaS (Portafolio)

**Autor:** Roman Velasco Moctezuma  
**Ubicación:** Guadalajara, Jalisco (Remoto global)

Landing page moderna estilo SaaS construida con React + Vite + Tailwind CSS. El objetivo es demostrar capacidades de **UX/UI**, **arquitectura frontend**, **copywriting con AIDA** y **calidad visual premium** para productos digitales.

---

## Vista general

Esta landing representa un producto real de startup con enfoque en conversión:
- Mensaje claro y jerarquía visual sólida.
- Diseño minimalista con detalles premium.
- Componentes reutilizables y estructura escalable.
- Animaciones sutiles con Framer Motion.
- SEO básico y accesibilidad inicial.

---

## Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion

## Deploy en GitHub Pages

Este proyecto ya está configurado para GitHub Pages usando `gh-pages`.

```bash
npm install
npm run deploy
```

Luego en GitHub:
- Settings → Pages
- Branch: `gh-pages` / `root`

Tu sitio quedará publicado en:  
`https://roman00120.github.io/react-studio/`

## Enfoque UX/UI

- **AIDA en el copy** para dirigir la narrativa: Atención → Interés → Deseo → Acción.
- **Minimalismo funcional**: menos ruido, más foco en lo importante.
- **Consistencia visual** con tipografías premium (Sora + Manrope), espaciado generoso y cards limpias.
- **Microinteracciones** suaves para dar sensación de producto vivo sin distraer.
- **Accesibilidad básica** con contrastes correctos y `aria-labels` en CTAs.
